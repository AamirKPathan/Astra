# ASTRA Markdown Syntax Rules

| Rule Category | Syntax Requirement | Example |
|--------------|-------------------|---------|
| Document Title | H1 + Bold | `# **EN‑000X – Title**` |
| Section Header | H2 + Bold | `## **1. Purpose**` |
| Subsection Header | H3 + Bold | `### **Overview**` |
| Bold Text | Use `**` | `**important**` |
| Italic Text | Use `*` | `*note*` |
| Inline Code | Use backticks | `` `README.md` `` |
| Code Block | Triple backticks | ```md\ncode here\n``` |
| Lists | Hyphen bullets | `- item` |
| Numbered Lists | Standard numbering | `1. First` |
| Metadata Fields | Bold label + plain text | `**Author:** Name` |
| Tables | Standard pipe format | `| A | B |` |
| Horizontal Rule | Three dashes | `---` |
| Links | Standard Markdown | `[text](url)` |
| Acronym Definition | Define on first use | `ASTRA Autonomous Flight System (AAFS)` |
| File/Directory Names | Inline code | `` `docs/` `` |
| Document Footer | Bold allowed | `## **Ad Astra Per Artem**` |
