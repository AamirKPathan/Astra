# 1. Purpose

This document defines terminology used throughout the ASTRA Autonomous Flight System project.

The glossary promotes consistent communication and interpretation of engineering documentation.

---

# 2. Definitions

| Term | Definition |
|------|------------|
| AAFS | ASTRA Autonomous Flight System |
| GCS | Ground Control Station |
| IMU | Inertial Measurement Unit |
| GNSS | Global Navigation Satellite System receiver |
| Telemetry | Data transmitted from the vehicle during operation |
| Guidance | Determines the desired flight path |
| Navigation | Estimates the vehicle's current state |
| Control | Commands actuators to follow the desired state |
| Sensor Fusion | Combining multiple sensor measurements to improve state estimation |
| PID | Proportional–Integral–Derivative control algorithm |
| FDIR | Fault Detection, Isolation, and Recovery |
| SIL | Software-in-the-Loop testing |
| HIL | Hardware-in-the-Loop testing |
| Digital Twin | A software model representing the physical vehicle |
| Mission | A complete operational flight from launch through recovery |
